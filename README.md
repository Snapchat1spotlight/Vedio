# 📸 کامێرای تیلیگرام - Telegram Camera Bot

![ئایکۆنی کامێرا](https://img.icons8.com/fluency/96/000000/video-call.png)

ئەم پڕۆژەیە وێبسایتێکی سادەیە کە ڕێگە بە بەکارهێنەران دەدات وێنەی کامێرای خۆیان بگرن و ڕاستەوخۆ بنێرن بۆ بۆتی تیلیگرام.

## 🚀 **تایبەتمەندییەکان**
- ✅ وێنەگرتن لە کامێرای وێب
- ✅ ناردنی ڕاستەوخۆ بۆ تیلیگرام
- ✅ پشتیوانی لە مایکرۆفۆن
- ✅ دواوەگەڕان لەگەڵ هەموو وێبگەڕە مۆدێرنەکان
- ✅ ڕووکاری بەکارهێنەری سادە و ڕێک

## 📋 **پێویستییەکان**
- بۆتێکی تیلیگرام (لە @BotFather دروستبکە)
- ئایدی چاتی تیلیگرام (لە @userinfobot وەریبگرە)
- هەر وێبگەڕێکی مۆدێرن (کرۆم، فایەرفۆکس، ئێج، سافاری)

## 🔧 **چۆنیەتی ڕێکخستن**

### 1️⃣ **دروستکردنی بۆت لە تیلیگرام**
1. بچۆ بۆ [@BotFather](https://t.me/botfather)
2. فەرمانی `/newbot` بنێرە
3. ناوێک بۆ بۆتەکەت هەڵبژێرە
4. **تۆکنەکە** وەربگرە (شتی وەک: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2️⃣ **وەرگرتنی ئایدی چات**
1. بچۆ بۆ [@userinfobot](https://t.me/userinfobot)
2. فەرمانی `/start` بنێرە
3. کلیک لە "Share Contact" بکە
4. **ئایدی ژمارەکە** وەربگرە (وەک: `123456789`)

### 3️⃣ **ڕێکخستنی کۆد**
لە فایلی `index.html`، ئەم دوو هێڵە بگۆڕە بە زانیارییەکانی خۆت:

```javascript
const BOT_TOKEN = 'تۆکنەکەت لێرە دابنێ';
const CHAT_ID = 'ئایدی چاتەکەت لێرە دابنێ';
```

### 4️⃣ **بڵاوکردنەوە لە گیتهەب**
1. ئەم فایلانە بخە ناو ڕێپۆیەکی گیتهەب
2. چالاککردنی **GitHub Pages**:
   - بچۆ بۆ Settings > Pages
   - لە "Branch" هەڵبژێرە `main`
   - کلیک لە Save بکە
3. پاش چەند خولەکێک، وێبسایتەکەت لەم ڕێگەیەوە دەست دەکەوێت:
   `https://[USERNAME].github.io/[REPO-NAME]/`

## 📱 **چۆنیەتی بەکارهێنان**
1. سەردانی وێبسایتەکە بکە
2. چاوەڕوانی لۆدبوونی پەڕەکە بە
3. کلیک لە **👍 ڕازیم** بکە بۆ ڕێگەپێدان بە کامێرا
4. وێنەیەک دەگیرێت و بۆ تیلیگرام دەنێردرێت
5. ئەگەر **👎 نارازیم** کلیک بکەیت، هیچ ڕوونادات
